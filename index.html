ustin Home Prices

Overview

This analysis examines house sale prices in Austin, Texas, spanning 2018 through 2021 in relation to 30 other variables, including number of baths and beds, parking spaces, land and construction area. We analyzed how these variables influenced sale prices. The variables were used as inputs for both a linear model and a neural network.

How to use
  • Click/Tap the markers on the map to see details about postings.
  • Add filters to reduce the amount of results on the map.
  • Choose a variable to compare the linear and neural network prices against the real one on a scatter plot.
Tools
  • Linear Regression - Scikit Learn
  • Neural network - TensorFlow
  • Plots - Plotly JS
  • Map - Leaflet JS
  • Layout - Bootstrap and CSS
