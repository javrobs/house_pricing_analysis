Austin Home Prices

Overview

We analyzed Austin, Texas, home prices for sales from 2018 through 2021. We used 30 variables as inputs for both a linear model and a neural network. We analyzed how these variables influenced sale prices. Variables included number of bathrooms and bedrooms, parking spaces, land area, and construction area.

Tools
  • Linear Regression - Scikit Learn
  • Neural network - TensorFlow
  • Plots - Plotly JS
  • Map - Leaflet JS
  • Layout - Bootstrap and CSS
