Overview

This is a real estate analysis focused on Austin, TX from 2018 to 2021. In this analysis, we have processed 30 variables which were subsequently used as input for both a linear model and a neural network. The primary objective is to delve into the factors influencing property sale prices. This examination encompasses various attributes, including but not limited to the count of bathrooms and bedrooms, available parking spaces, land area, and construction area.

Tools
  • Linear Regression - Scikit Learn
  • Neural network - TensorFlow
  • Plots - Plotly JS
  • Map - Leaflet JS
  • Layout - Bootstrap and CSS
